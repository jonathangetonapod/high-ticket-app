# Warmup Requirements

## Minimum Standards

- **Warmup Duration:** Minimum 14 days before launching campaigns
- **Warmup Score:** Minimum 50/100 to be considered healthy
- **Bounce Rate:** Must be below 5% (warning at 2%)

## Critical Blockers

These will **BLOCK** submission:

1. Any mailbox disabled for bouncing
2. Warmup score below 30
3. Bounce rate above 5%
4. Account created less than 7 days ago

## Warnings

These will show warnings but allow submission:

1. Warmup score between 30-50
2. Bounce rate between 2-5%
3. Less than 140 emails sent (less than 14 days warmup)
4. Warmup paused

## Notes for Strategists

- Always check mailbox health BEFORE starting campaign setup
- If any mailboxes are critical, contact client to resolve first
- Document any mailbox issues in the Loom video
