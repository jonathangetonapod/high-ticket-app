# Lead List Requirements

## Required Fields

Every lead MUST have:

1. **email** — Valid email address
2. **first_name** — First name for personalization
3. **company** — Company name
4. **title** — Job title (for ICP matching)

## Recommended Fields

- last_name
- linkedin_url
- company_size
- industry
- phone
- website

## ICP Matching

- Minimum **90% ICP match** required
- Leads outside ICP must be flagged and explained
- Title/seniority must match strategy call requirements
- Company size must be within target range

## Data Quality

- Email validation: 95%+ valid emails required
- No generic emails (info@, contact@, hello@)
- No competitor emails
- Deduplicated within list and against previous campaigns

## List Size Guidelines

- Minimum: 500 leads per campaign
- Maximum: 5,000 leads per campaign (for manageability)
- If under 500, combine with another ICP segment

## Blockers

Submission blocked if:

1. ICP match below 90%
2. Missing required fields
3. Email validation below 95%
4. Contains obvious bad data (test@, fake names)
